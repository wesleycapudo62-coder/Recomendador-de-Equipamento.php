# 📡 Recomendador de Equipamentos de Rede

## 📌 Descrição

Sistema simples em PHP que recomenda equipamentos de rede com base no tipo de ambiente escolhido pelo usuário.

---

## ⚙️ Funcionalidade

O usuário seleciona um ambiente através de um formulário HTML (método POST).
O sistema processa a escolha utilizando `switch` no PHP e exibe a recomendação adequada.

---

## 🖥️ Tecnologias utilizadas

* HTML
* PHP

---

## 📋 Ambientes disponíveis

* **Residencial** → Roteador Wireless (SOHO)
* **Escritório** → Switch Gerenciável de 24 portas
* **Datacenter** → Roteador de Borda + Switch Layer 3

---

## ⚠️ Validação

Caso nenhuma opção seja selecionada, o sistema exibe uma mensagem de erro.

---

## 🚀 Objetivo

Praticar:

* Formulários HTML (`POST`)
* Estrutura `switch/case`
* Validação de dados em PHP
